popularity*
用於排名藝術家相對於 Spotify 上其他藝術家的受歡迎程度上受歡迎程度的指標。 每兩天更新一次，因此新曲目可能會顯示 0%。
year
年份
Danceability*
根據節奏、節奏穩定性、節拍強度和整體規律性等音樂元素的組合來描述曲目是否適合跳舞。 值 0.0 表示最不適合跳舞，1.0 表示最適合跳舞。
Energy*
能量是從 0.0 到 1.0 的度量，代表強度和活動的感知度量。 通常，充滿活力的曲目讓人感覺快速、響亮且嘈雜。 例如，死亡金屬具有高能量，而巴赫前奏曲在音階上得分較低。 對該屬性有貢獻的感知特徵包括動態範圍、感知響度、音色、起始率和一般熵。
Key
曲目所在的調。整數使用標準音高等級表示法映射到音高。 例如。 0 = C、1 = C♯/D♭、2 = D，依此類推。 如果未檢測到鍵，則值為 -1。
loudness
曲目的整體響度，以分貝 (dB) 為單位。 響度值是整個音軌的平均值，對於比較音軌的相對響度非常有用。 響度是聲音的質量，是身體強度（振幅）的主要心理相關因素。 值通常介於 -60 和 0 db 之間。
Mode
調式表示曲目的模態（大調或小調），以及從中導出其旋律內容的音階類型。 大調用 1 表示，次要用 0 表示。
Speechiness*
言語能力檢測曲目中是否存在口語單詞。 錄音越像語音（例如脫口秀、有聲讀物、詩歌），屬性值越接近 1.0。 高於 0.66 的值描述的曲目可能完全由口語單詞組成。 0.33 到 0.66 之間的值描述可能同時包含音樂和語音的曲目，無論是分段還是分層，包括說唱音樂等情況。 低於 0.33 的值很可能代表音樂和其他非類似語音的曲目。
acousticness   X
音軌是否原聲的置信度，範圍為 0.0 到 1.0。 1.0 表示該曲目的原聲可信度很高,例如電子音。
instrumentalness*
預測曲目是否不包含人聲。 在這種情況下，“Ooh”和“aah”聲音被視為樂器聲音。 說唱或口語曲目顯然是“有聲的”。 器樂性值越接近 1.0，曲目不包含人聲內容的可能性就越大。 高於 0.5 的值旨在代表樂器曲目，但當值接近 1.0 時置信度會更高。
Liveness*
檢測錄音中是否有觀眾。 較高的活躍度值表示該曲目現場表演的可能性增加。 高於 0.8 的值表明該曲目很有可能是現場演唱的。
Valence
0.0 到 1.0 之間的衡量標準，描述曲目所傳達的音樂積極性。 高的曲目聽起來更積極（例如快樂、開朗、欣快），而低的曲目聽起來更消極（例如悲傷、沮喪、憤怒）。
Tempo
曲目的總體估計節奏（以每分鐘節拍數 (BPM) 為單位）。 在音樂術語中，節奏是給定樂曲的速度或步調，直接源自平均節拍持續時間。
Time_signature
估計的拍號。 拍號（米）是一種符號約定，用於指定每個小節（或小節）中有多少節拍。 拍號範圍從 3 到 7，表示拍號“3/4”到“7/4”。
Duration_ms
曲目的持續時間（以毫秒為單位）。
